# 🐊 Крокодил — кыргызча Telegram бот

Классикалык «Крокодил» сөз түшүндүрүү оюну. Топто ойнолот: бир ойунчу сөздү
көрөт да түшүндүрөт, калгандары чатта табышат. Туура тапкан адам жаңы жетекчи болот.

## Файлдар

| Файл | Милдети |
|------|---------|
| `main.py` | Боттун негизги логикасы (aiogram 3, polling) |
| `words.py` | ~200 кыргызча сөз, категориялар боюнча |
| `requirements.txt` | Python китепканалары |
| `Procfile` | Railway/Heroku иштетүү командасы |
| `railway.json` | Railway деплой конфигурациясы |
| `.env.example` | Токендин үлгүсү (жергиликтүү тест үчүн) |

## Railway'ге чыгаруу (деплой)

1. Бул файлдардын баарын GitHub репозиторийге жүктө (push).
2. Railway'де сервис ошол репого туташкан болсун (сенде даяр).
3. Railway → сервис → **Variables** → `BOT_TOKEN` бар экенин текшер
   (@BotFather берген токен). ✅ Сенде кошулган.
4. Railway автоматтык түрдө кайра деплой кылат. **Deployments** бөлүмүндө
   логду көрсөң: `Бот иштеп баштады (polling)...` деген жазуу чыгат.

> Токенди эч качан кодго жазба — ал Railway Variables ичинде гана турат.

## Ботту топко кошуу

1. Ботту Telegram тобуна кош.
2. @BotFather'да боттун **Group Privacy** орнотуусун **өчүр** (Disable) —
   ансыз бот топтогу бардык жазууларды көрбөйт, ойунчулардын жоопторун
   таба албай калат.
3. Топто `/game` деп жаз — оюн башталат.

## Командалар

- `/start` — кош келүү
- `/help` — эрежелер
- `/game` — жетекчи болуп оюнду баштоо
- `/stop` — раундду токтотуу

## Сөздөрдү салыштыруу эрежеси

Бот жоопту салыштырганда:

- чоң/кичине тамга **маанилүү эмес** — `УЙ` = `уй`
- `ө/ү/ң` тамгалары `о/у/н` катары да кабыл алынат
- дефис (`-`) менен боштук (` `) **маанилүү** — алар теңелбейт

**Мисал**, сөз `үй-бүлөө` болсо:

| Жооп | Натыйжа |
|------|---------|
| `үй-бүлөө` | ✅ туура |
| `уй-булоо` | ✅ туура |
| `УЙ-БУЛОО` | ✅ туура |
| `уй булоо` | ❌ туура эмес (дефистин ордуна боштук) |
| `үй бүлөө` | ❌ туура эмес (дефистин ордуна боштук) |

## Сөз кошуу

`words.py` ичиндеги тиешелүү категорияга сөздү кошуп, кайра push кыл —
Railway өзү жаңыртат.
